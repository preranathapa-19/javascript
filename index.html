"First, solve the problem. Then, write the code."

A function in JavaScript is a reusable block of code that performs a specific task. It helps to organize code and avoid repetition. We can call a function whenever we need to use that task in our program. A function can take inputs (called parameters), process them, and return an output (result). Functions make programs cleaner, more structured, and easier to manage.
